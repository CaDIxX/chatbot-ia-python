# Chatbot Inteligente com Python & OpenAI

Este projeto é um Chatbot interativo que utiliza a API do GPT-4o para responder perguntas em tempo real. Foi desenvolvido utilizando a biblioteca Streamlit para a interface web.

## Tecnologias
- **Python**: Lógica do sistema.
- **Streamlit**: Framework para criação da interface do usuário (UI).
- **OpenAI API**: Engine de inteligência artificial para processamento de linguagem natural.

## Funcionalidades
- Interface de chat amigável e moderna.
- Memória de contexto (o bot lembra o que foi dito na conversa).
- Integração direta com os modelos mais recentes da OpenAI.

## Como rodar o projeto
1. Clone o repositório.
2. Instale as dependências: `pip install streamlit openai`
3. Adicione sua chave da API da OpenAI no código.
4. Execute o comando: `streamlit run main.py`
